## Analytics Portfolio

**[Corporate Catering Operations — PostgreSQL](https://github.com/gonzaloguerreros/catering-operations-sql)**  
Built the schema from scratch, then four query sets: supplier delivery performance, revenue trends, account retention cohorts, promo ROI. The retention query flags accounts that have gone quiet relative to their own historical ordering cadence.

**[A/B Test Analysis — Python](https://github.com/gonzaloguerreros/catering-ab-test-python)**  
Full experiment analysis for a simulated discount campaign. Welch's t-test, power analysis, Bonferroni subgroup analysis. Result was "do not ship" — conversion lift wasn't detectable, but first-order spend was genuinely higher (+13%, p=0.004). 21 unit tests.

**[Review Intelligence — NLP Python](https://github.com/gonzaloguerreros/catering-review-nlp-python)**  
Scores 1,200 catering reviews with VADER + TextBlob, uses LDA to find what topics people are actually complaining about. Exports to Tableau. 17 unit tests.

**[Price Optimization — Monte Carlo](https://github.com/gonzaloguerreros/OJ_Optimization)**  
Graduate-level pricing project. Log-log regression for elasticity, then 10,000 Monte Carlo simulations to get a confidence band on the optimal price rather than a point estimate.
